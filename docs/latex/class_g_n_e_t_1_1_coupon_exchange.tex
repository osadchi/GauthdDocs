\doxysubsection{Detailed Description}
Класс для обработки RPC-\/запросов обмена купонов между клиентом и сервером. @inherits Rpc. 

Данный класс предоставляет методы для обработки запросов на сервере и клиенте, а также обработки таймаутов при ожидании ответа.
