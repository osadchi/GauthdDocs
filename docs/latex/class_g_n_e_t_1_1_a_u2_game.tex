\doxysubsection{Detailed Description}
Класс протокола для передачи данных между сервером аутентификации и игровым сервером. 

Наследует GNET\+::\+Protocol и содержит данные, специфичные для этого протокола. Используется для обработки запросов аутентификации, передачи информации о пользователях и других взаимодействий.
