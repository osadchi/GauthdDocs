\doxysubsection{Detailed Description}
Протокол для оповещения о скидках и специальных предложениях 

Наследуется от базового класса Protocol и используется для рассылки информации об изменениях в системе скидок между серверами и клиентами. Содержит данные о маркетинговых акциях и специальных предложениях от продавцов.
