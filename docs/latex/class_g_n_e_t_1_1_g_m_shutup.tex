\doxysubsection{Detailed Description}
Протокол блокировки чата пользователя 

Наследуется от базового класса Protocol. Используется для обработки команды принудительного отключения пользователя от чата (gag). Параметры протокола включаются через файл gmshutup и могут содержать идентификатор пользователя, длительность блокировки и другие параметры.
