\doxysubsection{Detailed Description}
Класс для обработки биллинг-\/запросов в системе. 

Наследуется от GNET\+::\+Protocol. Предназначен для управления запросами биллинга, такими как обработка платежей, проверка статуса и т.\+д. Содержит метод Process для обработки данных в контексте менеджера и сессии.
