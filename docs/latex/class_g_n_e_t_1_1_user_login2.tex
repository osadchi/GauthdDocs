\doxysubsection{Detailed Description}
Класс для обработки RPC-\/запросов авторизации пользователей 

Наследует функциональность Proxy\+Rpc и реализует обработку запросов входа пользователя, взаимодействие с базой данных и обработку результатов авторизации.
