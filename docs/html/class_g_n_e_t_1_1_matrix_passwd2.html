Detailed Description

Прокси-класс для обработки RPC-запросов проверки матричного пароля @inherits ProxyRpc.

Класс реализует:

  • Обработку доставки RPC-запросов
  • Постобработку результатов
  • Таймауты соединения (клиентская сторона)
