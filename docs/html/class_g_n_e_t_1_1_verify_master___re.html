Detailed Description

Класс протокола для обработки ответов верификации мастер-сервера

Наследует функциональность базового класса Protocol и реализует обработку ответов на запросы проверки мастер-сервера. Содержит данные протокола, автоматически генерируемые из описания в файле verifymaster_re.
