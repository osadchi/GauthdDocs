Detailed Description

Протокол для взаимодействия между игровым сервером и сервером аутентификации/учётных записей (Auth/AU Server)

Наследует базовый класс Protocol и содержит обработчик для входящих сообщений. Реализация протокола должна быть завершена в методе Process().
