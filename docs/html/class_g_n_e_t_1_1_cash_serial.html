Detailed Description

Прокси-класс для обработки RPC-запросов денежных серий

@inherits ProxyRpc

Note
Автоматически генерирует базовый RPC-функционал через макрос RPC_BASECLASS. Для работы требует подключения реализации cashserial (сгенерированной RPC-системой)
